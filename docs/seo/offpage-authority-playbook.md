# Off-Page SEO Authority Playbook (India First)

## 1) Google Business Profile (Week 1)
- Ensure NAP is exactly consistent with website footer/contact page.
- Set primary category: `Pharmaceutical equipment supplier`.
- Add secondary categories for machinery manufacturing and industrial equipment.
- Upload 20+ factory and machine images with descriptive file names.
- Publish one weekly update post linking to category/product pages.

## 2) Citation and Directory Build (Week 1-4)
- Submit business profile to India-first B2B directories and pharma trade directories.
- Keep naming consistent as `Kailash Machine Tools` and include legacy aliases in description.
- Always link back to key commercial pages:
  - `/pharmaceutical-machinery-manufacturer-ahmedabad-india`
  - `/vial-filling-machine-manufacturer-india`
  - `/ampoule-filling-sealing-machine-manufacturer-india`

## 3) Backlink Outreach (Week 3-10)
- Build a target list of:
  - Pharma media websites
  - Validation/compliance consultants
  - Packaging and process engineering blogs
  - Trade event websites and associations
- Offer technical guest content:
  - Vial line qualification checklist
  - Rotary vs linear washing comparison
  - cGMP design pitfalls in injectable machinery
- Require contextual do-follow links to relevant category/product pages.

## 4) Digital PR and Trust Signals (Week 5-12)
- Publish milestone and export announcements with machine photos.
- Add case-study snippets with client outcomes and commissioning context.
- Submit brand story updates using both current brand and legacy aliases.

## 5) Monthly Off-Page KPI Targets
- 20+ new quality citations per month (first 2 months).
- 5-8 new referring domains per month from relevant industry sites.
- Increase branded query impressions in Search Console by 20% month-over-month.
