# SEO Measurement Dashboard Setup

## Tracking Stack
- Google Search Console (query + indexing data)
- Google Analytics 4 (session + conversion data)
- Rank tracker (weekly position tracking for priority keywords)

## Environment Variables
Set these in production:
- `GOOGLE_SITE_VERIFICATION`
- `BING_SITE_VERIFICATION`
- `GOOGLE_ANALYTICS_MEASUREMENT_ID`

## Core KPIs (Weekly)
1. Clicks and impressions by top target queries
2. Average position for commercial keywords
3. CTR per landing page
4. Organic enquiries submitted from contact form
5. New vs returning organic users

## Priority Keyword Buckets
- Brand: `kailash machine tools`, `ganesh machine tools`, `shree ganesh pharmatech`
- Commercial: `pharmaceutical machinery manufacturer india`, `vial filling machine manufacturer india`, `ampoule filling machine manufacturer india`
- Supporting: `pharma washing machine`, `injectable liquid filling machine`, `capping and sealing machine`

## Monthly Review Workflow
1. Export Search Console query and page reports.
2. Identify pages ranking in positions 6-20.
3. Improve title/meta and first 200 words for those pages.
4. Add 2-4 internal links from high-authority pages.
5. Recheck performance after 14 days.
